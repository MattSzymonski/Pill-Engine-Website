## Quick start

Follow these commands to run an example locally.

```powershell
git clone https://github.com/MattSzymonski/Pill-Engine.git
cd Pill-Engine
cd docs
npm install
npm run docs:dev
```

To run an example game:

```bash
cd examples/Empty
cargo run
```
